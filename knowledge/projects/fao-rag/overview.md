# 法奥机器人智能客服平台概述

## 项目性质

- 南昌大学 × 法奥机器人校企合作
- 基于 AIFlowy 大型 AI 应用平台进行二次开发
- 不是从零独立开发整个 AIFlowy
- 当前简历角色：Agent 后端开发
- 时间：2026.01—2026.06

## 技术栈

- Java
- Spring Boot
- MyBatis-Flex
- Vue 3
- MySQL
- Redis
- Qwen
- SSE
- Docker
- Nginx

## 业务目标

面向机器人技术咨询、资料检索和售后服务，构建包含以下能力的客服工作台：

- 智能问答
- Agent 任务规划
- QA 数据沉淀
- 售后工单
- 管理端
- 用户端
- 私有化部署

## 项目定位说明

这是在法奥机器人与南昌大学的校企合作框架下进行的二次开发项目，不是正式实习。AIFlowy 是已有的开源 AI 应用平台，本项目在其基础上进行功能增改，朱旭的工作聚焦于 Agent 路由、规划、QA 数据闭环、工单和部署链路，而非重建整个平台。
