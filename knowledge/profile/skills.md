# 技术技能清单

## 主力技能（有项目实践支撑）

### 编程语言
- **Python**：主力语言，用于后端开发、脚本、AI 管道实现

### Web 后端框架
- **FastAPI**：async/await、路由、依赖注入、SSE 流式输出
- **SQLAlchemy**（async）：ORM 建模、关系映射、异步查询

### 数据库
- **PostgreSQL**：主要数据库，熟悉索引、约束、迁移
- **pgvector**：向量扩展，用于语义检索
- **Alembic**：数据库迁移管理

### AI / RAG 相关
- **RAG 管道实现**：文档解析、切块、向量化、检索、生成完整流程
- **sentence-transformers**：本地 Embedding 模型（bge-small-zh-v1.5）
- **向量检索**：余弦相似度、Top-K 检索、重排序
- **DeepSeek API**：流式对话、Prompt 工程、上下文管理
- **OpenAI API**：基本调用，接口格式兼容

### 前端（辅助）
- **Jinja2**：服务端模板渲染
- **HTMX**：无 JS 框架的动态交互、SSE 实时推送

### 工程化与部署
- **Docker / Docker Compose**：容器化部署，多服务编排
- **Nginx**：反向代理配置
- **Git**：版本控制，日常使用
- **Linux 基础**：命令行操作、进程管理、文件权限
