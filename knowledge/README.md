# 朱旭个人招聘知识库

本知识库以 `MASTER_ALL.md` 为唯一主版本，面向个人招聘网站、RAG 知识库、HR 问答和技术面试准备。

## 当前候选人定位

- 南昌大学软件工程专业，2027 届
- 主要求职方向：Agent 应用开发、RAG、AI 应用后端、Python 后端
- 正式实习：长冈医疗科技有限公司嵌入式软件工程师
- AI 实践：个人项目与法奥机器人校企合作
- 机器学习：仅在 Mood Tracker 中具备传统机器学习文本分类实践，不扩大为算法工程师能力

## 核心项目顺序

1. 智扫通机器人智能客服 Agent
2. 法奥机器人智能客服平台
3. 朱旭个人招聘知识 Agent
4. 情绪分析日记与 AI 心情助手

## 使用建议

- 完整导入：使用 `MASTER_ALL.md`
- 分项目切分：使用 `projects/` 目录
- HR 高频问答：使用 `06_hr_interview_qa.md`
- 技术追问：使用 `07_technical_interview_qa.md`
- 程序化入库：使用 `data/` 目录
- Agent 系统提示词：使用 `prompts/system_prompt.txt`

## 真实性原则

- 正式实习、校企合作和个人项目必须分开表述。
- Farino 是基于 AIFlowy 的二次开发，不描述为从零独立开发整个平台。
- 所有离线指标必须附带样本规模，不能等同于线上效果。
- Mood Tracker 的机器学习能力仅限传统文本分类项目实践。
- 不使用“精通”，不虚构用户量、商业收益、SLA 或生产规模。
- 其他不相关经历已移除。
